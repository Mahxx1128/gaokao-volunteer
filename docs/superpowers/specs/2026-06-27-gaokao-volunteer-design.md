# 高考志愿填报助手 — 设计文档

**日期:** 2026-06-27
**用户:** 山东高考女生，503分
**目标:** 纯前端单页工具，输入分数后用位次-线差混合法推荐可报考学校/专业

---

## 1. 产品定位

纯工具型单页应用。用户打开即用，输入分数、选科，查看全国可报考院校。实用第一，数据驱动。

## 2. 用户故事

1. 用户打开 `index.html`，看到中国地图，左侧是省份地图，右侧是空白面板
2. 顶部输入栏显示默认值：**山东省、503分、综合改革**
3. 用户可修改分数、选科，点击「分析」按钮触发预测
4. 地图以三色标注各省份的报考可行性（绿=安全、黄=冲刺、红=困难）
5. 点击省份 → 右侧面板显示该省各城市的学校和专业列表
6. 每所学校显示：录取概率百分比、学院、专业名称、往年分数线、就业方向标签
7. 支持「冲刺/稳妥/保底」三级筛选

## 5. 预测引擎算法

### 5.1 位次法（权重0.6）
1. 查询2026年一分一段表 → 503分对应位次 R_user
2. 对每个学校专业，查近5年录取位次 R_hist[i]
3. 计算位次偏差率：dev = mean((R_hist[i] - R_user) / R_user)
4. 映射为分数：在历年一分一段表中查找对应分数

### 5.2 线差法（权重0.4）
1. 2026年线差 Δ = 503 - 2026年批次线
2. 对历年：预测分 = 历年批次线 + Δ
3. 历年预测分取均值

### 5.3 混合计算
- 混合预测分 = 位次法预测分 × 0.6 + 线差法预测分 × 0.4
- 录取概率 P = sigmoid((混合预测分 - 用户分) / σ)
- P > 70% → 保底 | 40-70% → 稳妥 | < 40% → 冲刺

### 5.4 一分一段表（内置5年）
- 2021-2025年山东省一分一段表（分段累进）
- 2026年使用2025年数据按比例缩放

## 6. UI/UX 规范

- **配色:** 蓝色主调 #1a73e8，绿色 #34a853（保底），黄色 #fbbc04（稳妥），红色 #ea4335（冲刺）
- **字体:** 系统默认中文字体（苹方/微软雅黑）
- **响应式:** 桌面优先，最小宽度1200px
- **地图:** ECharts 中国地图，省份悬停高亮，点击选中
- **结果面板:** 右侧固定宽度380px，可滚动

## 7. 数据来源

优先搜集以下公开数据源：
- 山东省教育招生考试院：一分一段表、投档线
- 各高校招生网：历年各省录取分数线
- 中国教育在线、阳光高考：院校库、专业库
- 教育部学科评估结果：用于专业实力参考

数据覆盖约15-20省、每省10-40所学校、每校1-5个专业。

## 8. 非功能需求

- **性能:** 页面加载 < 3s（含ECharts CDN），预测计算 < 500ms
- **兼容:** Chrome/Firefox/Edge 最新版
- **离线:** 非必需（首次需网络加载CDN）
- **数据更新:** 修改 data.js 文件即可，无需改代码

## 9. 不在范围内

- 不包含用户系统/登录
- 不包含服务器端
- 不包含实时数据同步
- 不包含移动端原生适配（桌面网页可用）
- 不包含AI推荐/聊天功能
